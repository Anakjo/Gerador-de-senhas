# Gerador-de-senhas
Gerador de senhas seguras desenvolvido com JavaScript puro, permitindo a customização de comprimento e tipos de caracteres para aumentar a segurança digital.

Projeto desenvolvido como parte de treino de JavaScript.

# Funcionalidades
- Geração de senhas com comprimento customizável.
- Seleção de critérios (letras, números e símbolos).
- Interface responsiva e intuitiva.

# Tecnologias
- HTML5 (Estrutura semântica)
- CSS3 (Estilização e Flexbox)
- JavaScript (Lógica de geração aleatória e manipulação do DOM)

# Conceitos Aplicados
Este projeto aplica conceitos de **Engenharia de Software** como:
- Validação de inputs.
- Algoritmos de seleção aleatória (`Math.random`).
- UX (User Experience) com feedback visual.
